# Mama's 60th Birthday Gift 🎂✈️

A personal, math-themed digital gift card: a **€200 flight voucher** for our mom's 60th
birthday, wrapped in a riddle she has to solve first.

See [PLAN.md](PLAN.md) for the full concept, requirements and roadmap.

## What it does
1. Festive math-themed landing page (the number **60** is the motif).
2. A **riddle** she solves to unlock the gift.
3. A **"Redeem your gift"** button reveals a boarding-pass-style **€200 ticket**.
4. A short **form** for the flight details she's chosen — submitting emails them to us so
   we book the real tickets.

## Run it
Just open **`index.html`** in any browser (double-click it). No install, no server.

## Send it to her as a link
Pick one:
- **Netlify Drop** — drag this folder onto <https://app.netlify.com/drop> → instant link.
- **GitHub Pages** — push this repo, enable Pages, share the URL.
